# Portafolio Profesional

Ingeniera Civil Industrial con enfoque en análisis de datos, visualización y mejora de procesos, orientada a la toma de decisiones y a la generación de valor para el negocio.

## Sobre este portafolio
Este repositorio reúne proyectos académicos desarrollados durante mi proceso de formación, enfocados en análisis de datos, diseño de dashboards ejecutivos y comunicación de información para audiencias estratégicas.

Cada proyecto incluye documentación, evidencia visual y archivos técnicos que respaldan el trabajo realizado.
